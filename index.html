Questions? Answered.

What format are the templates in?

Pure HTML, CSS, and JavaScript files. No frameworks, no build tools, no dependencies. Open in any browser, edit in any code editor, deploy anywhere. Each template uses CSS custom properties (variables) so you can change colors and fonts in seconds.

What format are the prompt packs in?

Each pack includes Markdown files (.md) for easy copy-paste and beautifully styled HTML files you can print to PDF. Both formats are included in every purchase.

Can I use these for client work?

Yes. You get a full commercial license. Use the templates for your own sites or client projects. Use the prompts for your own work or to serve clients. The only restriction: don't resell the files themselves as a competing product.

Do I get updates?

Yes. All future updates to any product you purchase are included at no extra cost. Buy once, benefit forever.

What if the prompts don't work for me?

Every prompt is designed with specific anti-generic instructions built in — they tell the AI exactly what NOT to do (no filler, no clichés, no generic advice). If you're not satisfied, reach out and we'll make it right.

Which AI tools do the prompts work with?

All of them. The prompts are written for ChatGPT, Claude, Gemini, and any other LLM. They're tool-agnostic by design — good prompts work everywhere.
